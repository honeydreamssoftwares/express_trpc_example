
# TRPC Express Demo
 Demonstrates TRPC use in vanilla client


## Installation

Install TRPC Express Demo

```bash
  npm install
```
    

## Usage/Examples

Run server
```bash
npm run dev-server
```

Run client
```bash
npm run dev-client
```

Fiddle with TRPC from \client\client.ts and \server\server.ts
Types are in \server\approuter.ts
